# MVideo v1 - 基础版影视搜索播放平台

基于 PHP 开发的轻量基础版影视搜索与播放系统，主打核心观影功能，部署简单、界面简洁，适配多端使用，满足基础影视观看需求。

## ✨ 核心功能

- 多源影视搜索：支持配置多片源，覆盖电影、电视剧等主流影视资源

- 高清在线播放：支持跳转播放，展示影片详情（导演、类型、评分等）

- 播放记录管理：自动记录观看历史，支持续播、清空记录

- 明暗主题切换：支持手动切换/跟随系统主题，偏好自动保存

- 响应式适配：兼容桌面/移动设备，界面元素自适应调整

- 基础安全防护：浏览器验证，非浏览器访问自动跳转

- 个性化配置：支持自定义站点标题、描述、背景图等基础参数

## 📋 环境需求

- PHP 7.4+（推荐 8.0+），需开启 cURL 扩展

- 支持 Nginx/Apache/IIS 等主流 Web 服务器

- 浏览器：Chrome 60+/Firefox 55+/Safari 12+/Edge 79+

- 项目目录需具备读写权限（用于缓存与会话存储）

## ⚠️ 重要声明

本项目仅用于学习研究，禁止商业用途；不存储视频资源，所有内容来自第三方源站，用户需遵守当地法律法规。
