## 影视搜索

内置HLS广告切片过滤逻辑，尽可能减少视频播放中的广告干扰，提供更纯净的观影体验。聚合多个优质视频源，智能搜索，丰富的内容库满足不同用户的观影需求。完美适配手机、平板、电脑等各种设备，支持键盘快捷键操作，随时随地享受观影。PHP程序前端配置无需数据库，极简、快速搭建。

## 使用

将文件上传到你的服务器。
 - 入口地址 **域名/index.php**
 - 后台地址 **域名/admin.php** (密码:123456)

   **建议更改后台文件名称！**

## 功能

1. 多片源搜索
2. 影片详情展示
3. 支持多人观影
4. 深色/浅色主题切换
5. 响应式设计
6. 视频增强
7. 语音聊天
8. 智能广告过滤
